GDP Analysis

Real GDP

This first graph looks at the Real GDP over time from 1947 to July 2022. This is a classic economic example of a trending time series as Real GDP tends to increase over time exponentially. Here is what the data looks like:

rgdp <- read.csv("GDP.csv")

rgdp$DATE <- as.yearmon(rgdp$DATE)
rgdp.xts <- xts(rgdp$GDP, order.by = rgdp$DATE)

ts_plot(rgdp.xts,
        title = "Real GDP",
        Ytitle = "Billions of Dollars")

Since we know that Real GDP is trending exponentially, we can de-trend it by taking the log of it, which looks like this:

#Looks, its de-trended (mostly)
ts_plot(log(rgdp.xts),
        title = "Real GDP",
        Ytitle = "log of Billions of Dollars")

Percent Change in Real GDP

Another way we can de-trend Real GDP is by looking at the percent change in Real GDP, which looks like this:

rgdpc <- read.csv("GDPC1.csv")
rgdpc$DATE <- as.yearmon(rgdpc$DATE)
rgdpc.xts <- xts(rgdpc$GDP, order.by = rgdpc$DATE)
ts_plot(rgdpc.xts,
        title = "Percent Change in Real GDP",
        Ytitle = "Percent Change")

This data has now become stationary data as we have de-trended it, and we can do other time series analysis on it.

Traffic Deaths

Finally, the last type of data to look at is data with the pdq format, which deals with auto regressive data, data that trends with previous values of itself, and moving average data, which deals with average values of data wrapped around itself, and d deals with the amount of differentiation needed.

cars <- read.csv(file = "Cars.csv")
#Just some data manipulation to pass on data to dygraph
cars$Year <- yearmon(cars$Year)
k <- 3
#This creates the moving average through the rollmean function
cars$roll <- c(rep(NA,k-1),rollmean(cars$Total, k = k))
cars1 <- xts(cars[,2:3], order.by = cars$Year)
colnames(cars1) <- c("Crash Fatalities","Moving Average")

#Simple example of a Moving Average Graph in this case
#p = 0, d = 0, and q = k which is 3 from the rollmean funciton
dygraph(cars1) %>%
  dyRangeSelector() %>%
  dyCSS("dygraph.css")

This data is just a moving average data set, so p and d are equal to 0, while q is equal to 2 in this situation.
